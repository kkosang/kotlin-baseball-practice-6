# 숫자야구게임

## 🚀기능 요구 사항🚀

### Presentation

- `InputView`
  - [ ] 예측 숫자 입력요청 메시지를 출력한다.
  - [ ] 예측 숫자를 입력 받는다.
  - [ ] 재시작 안내 메시지를 출력한다.
  - [ ] 재시작 입력 받는다.
- `OutputView`
  - [ ] 게임시작 메시지를 출력한다.
  - [ ] 각 라운드마다 결과를 보여준다.
  - [ ] 게임종료 메시지를 출력한다.
- `에러 문구`
  - [ ] "[ERROR]" 형식으로 에러를 출력한다.
  - [ ] 입력 에러 `[ERROR] 유효하지 않은 날짜입니다. 다시 입력해 주세요.` 메시지를 보여준다.
    - [ ] 방문날짜를 잘못 입력한 경우

### Business Model

- `예측숫자 검증`
  - [ ] 숫자인지?
  - [ ] 범위안에 속하는지?
  - [ ] 3자리인지?
  - [ ] 중복된 숫자가 없는지?
  - [ ] 하나라도 `False` 인 경우 `IllegalArgumentException`
- `기능`
  - [ ] 컴퓨터 숫자 랜덤값 생성
  - [ ] 예측 숫자와 컴퓨터 숫자 비교
  - [ ] 재시작
